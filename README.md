# Online-Retail
# Project Overview

## *This project analyzes retail transaction data to uncover business insights related to revenue performance, customer behavior, and product trends. The analysis was conducted using SQL for data extraction and Python for data processing and visualization.*

**Objectives**

- Analyze total and monthly revenue trends

- Identify top-performing products

- Evaluate country-wise revenue contribution

- Perform RFM-based customer segmentation

- Generate actionable business insights

**Tech Stack**

- SQL (Data querying & aggregations)

- Python (Pandas, Matplotlib, Seaborn)

- Jupyter Notebook

- Data Cleaning & Feature Engineering

**Project Workflow**
**1️ Data Cleaning (Python)**

- Removed null values

- Converted date columns to datetime

- Created Revenue column (Quantity × UnitPrice)

**2️ SQL Analysis**

- Total Revenue Calculation

- Monthly Revenue Trends

- Top Products by Revenue

- Revenue by Country

- Customer-level Aggregations

**3️ RFM Customer Segmentation**

- Recency: Days since last purchase

- Frequency: Number of transactions

- Monetary: Total customer spending

- Segmented customers into value tiers

**4️ Data Visualization**

-Line chart for monthly revenue

-Bar chart for top products

- Horizontal bar for country analysis

- Customer segmentation distribution charts

**📈 Key Insights**

- A small percentage of customers contribute a large portion of revenue.

- Certain products consistently generate higher revenue.

- Revenue is concentrated in specific geographic regions.

- Customer segmentation reveals high-value and at-risk customers.
